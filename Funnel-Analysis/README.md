# E-commerce AARRR 분석 프로젝트

## 📊 프로젝트 개요

캐글의 이커머스 데이터를 바탕으로 AARRR 프레임워크를 적용한 대시보드를 구성하고, 지표 하락 구간을 식별하여 문제를 정의했습니다. 해당 프레임워크 중 Referral 단계의데이터가 존재하지 않아 분석 대상에서 제외하였으며, 이후 SQL 기반 데이터 분석을 통해 원인을 탐색하고 가설 수립과 검증을 통해 인사이트를 도출했습니다.

---

## ❓ 문제 정의

- 매출이 감소하는 이유는 무엇인가?
- **객단가**는 유지되었지만 **주문 수**가 감소
---

## 🔍 분석 과정 요약

1. **매출 감소 요인 분석**
   - 주문 수/객단가 추이 분석
2. **신규 vs 기존 고객 비교**
   - 신규 유저와 기존 유저의 주문 수 분석
3. **고객 세그먼트 분석**
   - 가입 연차, 연령대, 혜택 여부별로 기존 고객 주문 수 비교
4. **고객 이탈 분석**
   - 월별 이탈률 (Churn Rate) 분석
5. **퍼널 분석**

- **Looker Studio**: 대시보드 시각화
